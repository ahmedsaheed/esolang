# Esolang REPL - Help

## Commands

- `.help` - Show this message
- `.exit` - Exit the repl
- `.clear` - Clear the screen
- `.about` - Show information about the repl
- `.version` - Show the version of the repl
- `.license` - Show the license of the repl

## Features

- Syntax highlighting
- Error handling
- REPL history
- Multi-line input

## Keybindings

- `Ctrl + C` - Exit the repl
- `Ctrl + L` - Clear the screen
